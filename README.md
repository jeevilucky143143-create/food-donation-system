# 🍽️ Food Donation System

A web app built using **Streamlit (Frontend)** + **Flask (Backend)** that connects **Donors** and **NGOs** to reduce food waste.  
Donors can **register, post food donations with images and live location**, while NGOs can **browse and claim available donations**.  

---

## 🚀 Features
- 🔐 Donor & NGO registration  
- 📸 Donors can upload food images  
- 📍 Donors can capture their **current location** (latitude & longitude)  
- 📝 Post donations with description and quantity  
- 🏢 NGOs can browse and claim donations  
- 🎨 Stylish Streamlit frontend with modern UI  

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit  
- **Backend:** Flask (Python)  
- **Database:** SQLite (default, can be changed to MySQL/Postgres)  
- **Extra:** `geocoder` for live location
